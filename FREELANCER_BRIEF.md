# Freelancer Deployment Brief: Vanyara Resort Website

We need this Next.js website deployed and made production-ready.

## Scope
- Deploy website on Vercel.
- Connect GitHub repo.
- Connect domain.
- Make admin panel functional with CMS/database.
- Keep design intact.
- Ensure mobile responsiveness.
- Connect all enquiry/reservation/event/career forms.

## CMS Requirements
Admin must allow non-technical team to edit:
- Text
- Images
- Videos
- Brochure PDFs
- Layout downloads
- Suites
- Experiences
- Residences
- Amenities
- Footer/social links
- Booking details

Recommended CMS: Sanity, Supabase, Firebase or Strapi.

## Form Requirements
All forms should send data to:
- Email
- Google Sheet or CRM

Forms:
- Reservation request
- Residence enquiry
- MICE proposal
- Event booking
- Career application
- General enquiry

## Deliverables
- Live website URL
- GitHub repo access
- Admin login
- CMS login
- Vercel access
- Short Loom/video showing how to edit website
- Deployment notes

## Quality Expectations
- Luxury resort quality
- Fast loading
- Responsive on mobile/tablet/desktop
- SEO ready
- Clean code
- No template-looking design
