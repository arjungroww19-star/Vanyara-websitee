# Vanyara Resort Website

Ultra-luxury eco wellness resort website concept for VANYARA, Dudhwa.

## Included
- Next.js website
- Tailwind CSS styling
- Website pages: Home, Suites, Wellness, Experiences, Residences, Location, Reservations, MICE, Events, Careers, Enquire
- Admin preview panel for editing content without coding
- Residences real estate section with brochure, walkthrough, layouts, amenities and enquiry

## Important note
The current Admin panel works as a front-end preview. Edits are stored only in browser state while the page is open. For production, connect this admin to a real CMS/database like Sanity, Supabase, Firebase or Strapi.

## Recommended production stack
- Next.js
- Vercel
- Sanity CMS or Supabase
- Resend/Formspree/Google Sheets for enquiries

## How to run locally
1. Install Node.js latest LTS.
2. Open terminal inside this folder.
3. Run:

```bash
npm install
npm run dev
```

4. Open: http://localhost:3000

## How to deploy on Vercel
1. Upload this folder to a GitHub repo.
2. Go to Vercel.com.
3. Import the GitHub repo.
4. Click Deploy.
5. Add custom domain under Project Settings > Domains.
